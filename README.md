# finanzas-app

Sistema personal de finanzas. Regla 10/10/10/70, multi-cuenta, presupuestos,
préstamos y metas. COP.

**Stack:** Supabase (Postgres + Auth + RLS) · HTML/CSS/JS vanilla con módulos ES
(esm.sh, sin build) · Cloudflare Workers (assets estáticos) · deploy automático por Git.
